# 后台管理 — 产品重量标签 API 接口文档

## 1. 产品列表 GET /api/product/list

### 请求参数

| 参数 | 类型 | 必填 | 说明 | 默认值 |
|---|---|---|---|---|
| pageNum | int | 否 | 页码 | 1 |
| pageSize | int | 否 | 每页条数 | 15 |
| keyword | string | 否 | 模糊搜索（仅匹配产品名称） | null |

### 响应示例

```json
{
  "code": 200,
  "msg": "success",
  "data": {
    "records": [
      {
        "id": "1",
        "name": "农家土鸡蛋",
        "price": 1.50,
        "stock": 100,
        "status": "已上架",
        "image": "/uploads/images/xxx.jpg",
        "uploadTime": "2026-05-27 14:30",
        "weightText": "142斤",
        "productType": "实物"
      }
    ],
    "total": 50,
    "pageNum": 1,
    "pageSize": 15,
    "pages": 4
  }
}
```

### weightText 字段说明

| 字段 | 类型 | 说明 | 示例 |
|---|---|---|---|
| weightText | string? | 重量标签文本，直接取自 `commodity.weight_text` | "142斤"、"5kg"、"1箱" |

## 2. 产品详情 GET /api/product/detail

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|---|---|---|---|
| id | int | 是 | 产品ID |

### 响应示例

```json
{
  "code": 200,
  "msg": "success",
  "data": {
    "id": "1",
    "name": "农家土鸡蛋",
    "price": 1.50,
    "stock": 100,
    "status": "已上架",
    "image": "/uploads/images/xxx.jpg",
    "coverImage": "/uploads/images/xxx.jpg",
    "carouselMedia": [
      { "type": "image", "url": "/uploads/images/carousel1.jpg" }
    ],
    "weightText": "142斤",
    "storageCondition": "常温保存",
    "specImages": ["/uploads/images/spec1.jpg"],
    "description": "新鲜农家散养土鸡蛋",
    "uploadTime": "2026-05-27 14:30",
    "productType": "实物"
  }
}
```

### weightText 字段说明

| 字段 | 类型 | 说明 | 示例 |
|---|---|---|---|
| weightText | string? | 重量标签文本 | "142斤" |
| storageCondition | string? | 保存条件 | "常温保存"、"冷藏保存" |
| description | string? | 产品描述（来自 spec_description） | "新鲜农家散养土鸡蛋" |
| specImages | string[] | 规格图片列表（最多5张） | ["/uploads/...jpg"] |
| productType | string? | 产品类型 | "实物"、"虚拟" |

---

## 3. 创建产品 POST /api/product/add

支持 `multipart/form-data`（传文件）和 `application/json` 两种格式。

### 请求参数

| 参数 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| name | string | 是 | 产品名称 | "农家土鸡蛋" |
| price | decimal | 是 | 产品价格 | 1.50 |
| stock | int | 否 | 库存 | 100 |
| status | string | 否 | "已上架" / "已下架" | "已上架" |
| coverImage | string | 否 | 封面图URL | "/uploads/xxx.jpg" |
| carouselMedia | object[] | 否 | 轮播图/视频 [{type, url}] | - |
| specImages | string[] | 否 | 规格图片URL列表 | - |
| unitId | int | 否 | 单位ID | 1 |
| **weightText** | **string** | **否** | **重量标签文本，直接写入 commodity.weight_text** | **"142斤"** |
| storageCondition | string | 否 | 保存条件 | "常温保存" |
| description | string | 否 | 产品描述 | "新鲜农家散养土鸡蛋" |
| productType | string | 否 | 产品类型 | "实物" |

### 响应

```json
{
  "code": 200,
  "msg": "success",
  "data": { "id": 1 }
}
```

---

## 4. 编辑产品 POST/PUT /api/product/edit

支持 `multipart/form-data` 和 `application/json`。

### 请求参数（在创建参数基础上额外增加）

| 参数 | 类型 | 必填 | 说明 |
|---|---|---|---|
| id | int | 是 | 产品ID |

其余参数同创建产品。

### 响应

```json
{
  "code": 200,
  "msg": "success",
  "data": "编辑成功"
}
```

### 错误码

| code | msg | 说明 |
|---|---|---|
| 400 | 参数不能为空 | 缺少必填参数 |
| 404 | 产品不存在或已被删除 | ID无效 |
| 500 | 创建/编辑失败：{原因} | 服务端异常 |

## 8. 获取重量标签下拉列表 GET /api/product/weight-tags

> 从 `unit` 表获取已启用的单位列表，供新增/编辑产品时下拉选择重量单位。

### 请求参数

无

### 响应示例

```json
{
  "code": 200,
  "msg": "success",
  "data": [
    { "label": "斤", "weightUnit": "斤" },
    { "label": "个", "weightUnit": "个" },
    { "label": "箱", "weightUnit": "箱" },
    { "label": "kg", "weightUnit": "kg" }
  ]
}
```

### 返回字段说明

| 字段 | 类型 | 说明 |
|---|---|---|
| label | string | 单位名称，如 "斤"、"个"、"箱" |
| weightUnit | string | 重量单位（同 label） |

### 数据来源

`unit` 表，`is_enabled = 1`（已启用），按 `unit_id` 排序。

---

## 重量字段数据流

```
                    数据库                         API 返回
                     ┌─────────────────┐
  Create/Edit ──────→│ commodity表     │──→ 直接返回
  weightText="142斤" │ weight_text     │     weightText: "142斤"
                     │ "142斤"         │
                     │                 │
                     │ unit_id → unit  │──→ unitName ―→ 前端下拉单位列表
                     │ .unit_name      │
                     └─────────────────┘
```

> **注意：** `weight_text` 是纯文本字段，存什么就展示什么，不做数值拆分。
