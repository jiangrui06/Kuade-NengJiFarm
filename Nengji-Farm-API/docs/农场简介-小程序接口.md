# 农场简介 - 小程序 API 接口文档

**接口：** `GET /api/farm/intro`

**认证方式：** 无需 Token（公开接口）

**响应格式：** 统一使用 `ApiResult` 包装

```json
{ "code": 0, "message": "success", "data": { ... } }
```

---

## 响应示例

```json
{
  "code": 0,
  "message": "success",
  "data": {
    "name": "能记家庭农场",
    "mainImage": "/api/file/image/xxx.jpg",
    "introduction": "我们的农场位于风景秀丽的乡村...",
    "philosophy": "我们秉承自然、健康、可持续的发展理念...",
    "contact": {
      "address": "江苏省南京市溧水区能记农场",
      "phone": "138-1234-5678",
      "email": "info@nengjifarm.com"
    },
    "features": [
      { "id": 1, "name": "", "image": "/api/file/video/xxx.mp4" },
      { "id": 2, "name": "", "image": "/api/file/video/yyy.mp4" },
      { "id": 3, "name": "", "image": "/api/file/video/zzz.mp4" }
    ]
  }
}
```

## 响应字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| name | string | 农场名称 |
| mainImage | string | 农场主图/农场图片 |
| introduction | string | 农场简介 |
| philosophy | string | 农场理念 |
| contact | object | 联系方式（含 address/phone/email） |
| features | array | 特色视频列表（最多 3 条） |

### features 子字段

| 字段 | 类型 | 说明 |
|------|------|------|
| id | long | 视频 ID |
| name | string | 预留字段（当前返回空字符串） |
| image | string | 视频地址 |

## 数据来源

| 响应字段 | 数据来源 | 说明 |
|---------|---------|------|
| name | sys_config → farm_name | 农场名称 |
| mainImage | 优先 sys_config → farm_image | 农场图片路径 |
|  | 降级 carousel 表 position="home" 排序首图 | 轮播图首图 |
|  | 再降级 videos 表排序首个视频 | 视频首帧 |
| introduction | sys_config → farm_introduction | 农场简介文本 |
| philosophy | sys_config → farm_philosophy | 农场理念文本 |
| contact | sys_config → farm_contact | JSON 解析为 {address, phone, email} |
| features | videos 表，按 sort_order 排序取前 3 条 | 特色视频 |

**mainImage 优先级逻辑：**

1. `farm_image` 配置值非空 → 直接使用
2. 轮播表（`carousel`）`position="home"` 排序第一张图
3. 视频表（`videos`）排序第一个视频地址

## 注意事项

- 公开接口，无需 Token
- `features` 固定返回最多 3 条
- `contact` 字段在 JSON 解析失败时返回空对象 `{address:"", phone:"", email:""}`
