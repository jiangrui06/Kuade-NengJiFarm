# 活动列表分页懒加载 — 接口文档

> 版本：v1.0 | 2026-05-29

---

## 接口

**GET** `/api/activity/list`

## 请求参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| page | int | 否 | 1 | 页码，从 1 开始 |
| pageSize | int | 否 | 10 | 每页条数 |
| keyword | string | 否 | - | 搜索关键词，按活动标题模糊匹配 |

## 响应

```json
{
  "code": 0,
  "message": "success",
  "data": {
```

## 响应字段说明

### data 层

| 字段 | 类型 | 必返 | 说明 |
|------|------|------|------|
| activities | object[] | 是 | 当前页活动列表 |
| total | int | 是 | 活动总数（前端据此判断是否有更多页） |
| page | int | 是 | 当前页码 |
| pageSize | int | 是 | 每页条数 |

### activities[]

| 字段 | 类型 | 必返 | 说明 |
|------|------|------|------|
| id | int | 是 | 活动 ID |
| title | string | 是 | 活动标题 |
| image | string | 否 | 活动封面图 URL |
| desc | string | 否 | 活动简介 |
| startDate | string | 是 | 开始日期 `yyyy-MM-dd` |
| endDate | string | 是 | 结束日期 `yyyy-MM-dd` |
| status | string | 是 | 固定返回 `"active"` |
| price | float | 是 | 活动价格 |
| categoryName | string | 否 | 分类名称 |
| date | string | 是 | 开始日期（前端展示用） |
| location | string | 否 | 活动地点 |
| participants | int | 是 | 已参加人数 |
| remainingSlots | int | 是 | 剩余名额 |
| duration | int | 是 | 核销有效期（天） |

## 分页逻辑

前端判断是否有更多页：`page * pageSize < total`

示例：
- `page=1, pageSize=10, total=25` → `1*10 < 25` → 有更多页
- `page=3, pageSize=10, total=25` → `3*10 >= 25` → 无更多页

## 排序

按活动创建时间倒序（最新活动在前）。

## 搜索

传 `keyword` 时按 `title LIKE '%keyword%'` 过滤。
