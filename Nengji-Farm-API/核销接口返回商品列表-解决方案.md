## 需要修改的内容

### 修改范围

仅 `Controllers/StaffVerifyController.cs`，集中在以下两个位置：

1. **`VerifyHistoryItem` 类** — 新增 `Items` 属性
2. **`GetCommodityVerifyHistoryAsync` 方法** — 为每条记录填充商品明细
3. **`VerifyHistoryItem.ToResponse()`** — 输出 `items` 字段

#### 3. GetCommodityVerifyHistoryAsync 填充 Items

现有代码中已经通过 `orderDetails` 字典加载了全部商品明细（按 `OrderId` 分组）：

```csharp
var orderDetails = records
    .GroupBy(x => x.o.OrderId)
    .ToDictionary(g => g.Key, g =>
    {
        var details = g.Where(x => x.d != null).Select(x => x.d!).DistinctBy(x => x.CommodityOrderDetailsId).ToList();
        var names = details.Select(x => x.GoodsName).Where(n => !string.IsNullOrWhiteSpace(n)).Distinct().ToList();
        var goodsName = names.Count > 0 ? string.Join("、", names) : null;
        var description = details.Select(x => x.GoodsName).FirstOrDefault(n => !string.IsNullOrWhiteSpace(n));
        return (goodsName, description);
    });
```

在此字典基础上，新增一个 `items` 版本，按 OrderId 返回完整明细列表：

```csharp
// 新增：按 OrderId 构建完整 items 列表
var orderItemsMap = records
    .GroupBy(x => x.o.OrderId)
    .ToDictionary(g => g.Key, g =>
    {
        return g
            .Where(x => x.d != null)
            .Select(x => x.d!)
            .DistinctBy(x => x.CommodityOrderDetailsId)
            .Select(d => (object)new
            {
                name = d.GoodsName,
                image = MediaHelper.NormalizeImageUrl(d.ImageUrl),
                quantity = d.Quantity,
                price = d.UnitPrice
            })
            .ToList();
    });
```
