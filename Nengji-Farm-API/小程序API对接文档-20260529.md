## 1. 活动详情多视频支持

### 接口说明

`GET /api/activity/detail?id={activityId}` 返回新增 `videos` 数组字段，包含活动关联的所有视频地址。

### 期望响应

```json
{
  "code": 0,
  "msg": "success",
  "data": {
    "id": 24,
    "title": "亲子农耕体验",
    "image": "/api/file/image/activity_cover.jpg",
    "images": ["/api/file/image/activity1.jpg", "/api/file/image/activity2.jpg"],
    "videos": [
      "/api/file/video/5e89109bd5cf407dbacba8e29fc02816.mp4",
      "/api/file/video/abc123def456.mp4"
    ],
    "video": "/api/file/video/5e89109bd5cf407dbacba8e29fc02816.mp4",
    "desc": "周末带孩子体验农耕乐趣",
    "date": "2026-06-01",
    "location": "稻田时光农场",
    "price": 99.0,
    "stock": 50,
    "remainingSlots": 50,
    "participants": 0,
    "duration": 30,
    "content": "详细活动说明……"
  }
}
```

### 字段说明

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `videos` | `string[]` | 否 | 所有视频文件路径数组，无视频时返回 `[]` |
| `video` | `string` | 否 | 保留字段，返回第一个视频地址（兼容旧版） |
| `image` | `string` | 否 | 活动封面图（已修复：不再错误返回视频地址） |

### 前端兼容逻辑

```
GET /api/activity/detail
  → data.videos 存在且 length > 0
       → 遍历 videos 数组渲染视频轮播
  → data.videos 为空或不存在
       → 降级：使用 data.video 生成单个视频（现有逻辑）
```

## 2. 商品订单确认收货

### 接口说明

`POST /api/order/{orderNo}/confirm`

| 项目 | 说明 |
|------|------|
| 请求方式 | POST |
| 路径 | `/api/order/{orderNo}/confirm` |
| 请求体 | 无（空对象 `{}`） |
| 需要登录 | 是（需 wx-token） |

### 正常响应

```json
{
  "code": 0,
  "msg": "success",
  "data": {
    "orderId": "ORD20260528123456789",
    "id": "ORD20260528123456789",
    "status": "completed",
    "statusText": "已完成",
    "completeTime": "2026-05-29 10:30:00"
  }
}
```

### 业务逻辑

1. 校验订单是否存在
2. 校验订单类型为商品订单（goods）
3. 校验当前状态为 `shipping`（运输中），非此状态拒绝
4. 校验订单归属权（只能确认自己的订单）
5. 将状态更新为 `completed`（已完成）
6. 记录完成时间（当前服务器时间）
7. 发放订单积分

### 错误码

| code | msg | 说明 |
|------|-----|------|
| 404 | 订单不存在 | orderNo 无效 |
| 400 | 该订单类型不支持确认收货 | 非商品订单 |
| 400 | 订单状态不允许确认收货 | 非 `shipping` 状态 |

### 前端调用方式

```javascript
wx.request({
  url: `https://api.nengjifarm.com/api/order/${orderNo}/confirm`,
  method: 'POST',
  header: { 'wx-token': token },
  success(res) {
    if (res.data.code === 0) {
      wx.showToast({ title: '收货成功', icon: 'success' });
      // 刷新订单详情
    } else {
      wx.showToast({ title: res.data.msg, icon: 'none' });
    }
  }
});
```
