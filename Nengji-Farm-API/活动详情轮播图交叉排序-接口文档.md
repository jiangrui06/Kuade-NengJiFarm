# 活动详情轮播图交叉排序 — 接口文档

> 版本：v1.0 | 2026-05-29

---

## 接口

**GET** `/api/activity/detail?id={id}`

## 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| id | int | 是 | 活动 ID |

## carouselMedia 字段

### 定义

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| type | string | 是 | 媒体类型：`"image"` 或 `"video"` |
| url | string | 是 | 媒体文件路径 |

### 数据来源

| 来源 | MaterialType | carouselMedia type | 说明 |
|------|-------------|-------------------|------|
| 轮播图 | 0 | `"image"` | 后台配置的轮播图片 |
| 后台视频 | 2 | `"video"` | 后台管理上传的视频 |
| 小程序视频 | null | `"video"` | 小程序端上传的视频 |
| 活动封面视频 | 活动表 `video_url` | `"video"` | 去重后插入数组头部 |

### 排序规则

按 `activity_material.sort_order` 升序遍历素材，图视混合排列。活动自身 `video_url` 去重后置于数组最前。

## 兼容性

| 字段 | 说明 |
|------|------|
| `carouselMedia` | **新增**，前端优先使用，严格按后端顺序渲染 |
| `images` | **保留**，前端降级使用（carouselMedia 不存在时） |
| `videos` | **保留**，兼容旧版 |
| `video` | **保留**，兼容旧版单视频地址 |
