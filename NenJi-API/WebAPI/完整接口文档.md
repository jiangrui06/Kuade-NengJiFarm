# 能记农场小程序后端完整接口文档

更新时间：2026-05-06  
基础地址：`http://192.168.203.56`

## 通用说明

### 返回格式

大部分接口返回：

```json
{
  "code": 0,
  "message": "success",
  "data": {}
}
```

`code = 0` 表示成功，非 0 表示失败。文件、图片、视频接口会直接返回文件流或静态资源内容。

### 认证方式

需要登录的接口请求头：

```http
Authorization: Bearer {token}
```

`token` 来自微信登录、微信手机号快捷登录等登录接口返回值。

### 本次新增/修复相关说明

- 首页搜索：`GET /api/home/search`，支持商品、活动搜索。
- 点餐下单：`POST /api/OrderDetails/create`，`sourceType=food` 时使用餐桌号 `tableNumber`，当前数据库已补齐 1-8 号桌。
- 订单列表：小程序订单列表建议使用 `GET /api/orders`，不是旧的 `GET /api/OrderDetails` 列表接口。
- 活动详情：`GET /api/activity/detail?id=1`，正确详情路由是 query 参数 `id`，视频地址在返回数据 `data.video`。
