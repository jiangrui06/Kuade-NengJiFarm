# API 变更说明：商品规格/描述字段拆分

## 变更原因

原 `description` 字段同时包含规格和描述（如 `"500g/份，脆甜多汁"`），现拆分为 `spec` + `description` 两个字段。

## 涉及接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/goods/detail?id={id}` | GET | 商品详情 |
| `/api/goods/list` | GET | 商品列表 |
| `/api/goods/search` | GET | 商品搜索 |
| `/api/farm-goods` | GET | 农场商品列表 |
| `/api/farm-goods/index` | GET | 农场首页 |
| `/api/farm-goods/category` | GET | 分类商品 |
| `/api/order/getGoodsList` | GET | 菜单商品列表 |
| `/api/order/getOrderData` | GET | 下单页数据 |
| `/api/home/search` | GET | 全局搜索 |

## 字段变化

```
// 旧响应
{
  "description": "500g/份，脆甜多汁",
  ...
}

// 新响应
{
  "spec": "500g/份",          // 新增：净含量+单位
  "description": "脆甜多汁",   // 变更：仅描述文本，不含规格
  ...
}
```

## 字段取值对照

| 商品示例 | 旧 description | 新 spec | 新 description |
|---------|---------------|---------|---------------|
| 甜养玉米500g | `500g/份，软糯香甜` | `500g/份` | `软糯香甜` |
| 红富士苹果500g | `500g/份，脆甜多汁` | `500g/份` | `脆甜多汁` |
| 散养土鸡蛋6枚 | `6枚/盒，林下散养` | `6枚/盒` | `林下散养` |
| 东北黑木耳200g | `200g/袋，肉质厚实` | `200g/份` | `肉质厚实` |
| 生态草莓礼盒 | `礼盒装，应季生态草莓` | `1盒/盒` | `礼盒装，应季生态草莓` |
| 有机蔬菜认购田 | `认购农场有机蔬菜田...` | _(空)_ | `认购农场有机蔬菜田...` |

## 注意事项

- `description` 字段**仅**包含描述文本，不含规格部分
- 原 `description` 用于显示规格的地方，请改为使用 `spec` 字段
- 无规格的商品（如认购田），`spec` 返回空字符串
- 菜品（dish）和活动（activity）接口不受影响，无此变更
